# 💀 Phantom Root — Nothing is Real

An advanced CTF web exploitation challenge.

## Setup

```bash
npm install
npm start
```

The challenge runs on port 3000 by default.  
Set `PORT` env var to override.

## Free Hosting (Render)

1. Push this folder to a GitHub repo
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your repo
4. Build command: `npm install`
5. Start command: `node server.js`
6. Done — share the URL with players
